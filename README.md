🌿 Profylles – Site Vitrine
📌 Description

Profylles est un site vitrine présentant une méthode d’accompagnement dédiée aux dirigeants et commerciaux, centrée sur la connaissance de soi et l’efficacité commerciale.

Ce projet a pour objectif de :

Présenter la méthode Profylles
Mettre en avant les offres d’accompagnement et de formation
Générer des prises de contact qualifiées

🚀 Installation
Cloner le projet :
git clone https://github.com/ton-repo/profylles.git
Ouvrir le projet :
cd profylles
Lancer en local :
Ouvrir index.html directement
ou utiliser une extension type Live Server
